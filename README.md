# todo-app-mern

## clone the project
```bash
git clone git@github.com:Ranganathb082/todo-app-mern.git
```

## Run the project
go to the directory
```bash
cd todo-app-mern
```
install all the packages
```javascript
npm i
```
to start the express server
```javascript
 npm run dev
```

go to [local](http://localhost:3000/)http://localhost:3000/
